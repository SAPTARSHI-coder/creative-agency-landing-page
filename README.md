# Creative Agency — Landing Page

A minimal landing page for a fictional creative design agency, built with pure HTML and CSS. The layout features a two-column card design with a branded heading and Google Fonts integration.

## Overview

The page includes a logo, a hero heading with an inline color-accented word ("Creative"), two side-by-side service cards (Beauty and Construction), and a footer tagline. Layout is handled with CSS Flexbox and absolute positioning.

## Tech Stack

| Technology | Role |
|---|---|
| HTML5 | Page structure |
| CSS3 | Layout and typography |
| Google Fonts (Poppins) | Heading and body typography |

## Project Structure

## Getting Started

Open `index.html` in any browser. An internet connection is needed to load the Google Font.

## Key Concepts

- Two-column card layout using **CSS Flexbox**
- Inline `<span>` styling for typography color accents
- Image sizing within card containers
- Google Fonts integration via `<link>` tag
